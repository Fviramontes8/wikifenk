Installing Executorch from source

Installing executorch v0.2.0 

git clone https://github.com/pytorch/executorch -b v0.2.0

Updating and pulling submodules 

cd executorch
      git submodule sync
      git submodule update --init

Upgrading setuptools

On Pop!_OS 22.04 it is required to upgrade setuptools before running install_requirements.sh. I do not know if this is needed for other distributions 

pip install -U setuptools

Installing requirements for executorch 

./install_requirements.sh
