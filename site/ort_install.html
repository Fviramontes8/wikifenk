Installing ONNX Runtime from source

Cloning repo and checking out version 1.15.0

To look at other available versions click here

git clone https://github.com/microsoft/onnxruntime -b v1.15.0
cd onnxruntime

Build flags to generate shared libraries for CPU only inference applications

./build.sh --config RelWithDebInfo --build_shared_lib --parallel --compile_no_warning_as_error --skip_submodule_sync

Installing shared objects and header files 

cd build/Linux/RelWithDebInfo
sudo make install
