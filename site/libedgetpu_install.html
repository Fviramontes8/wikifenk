Downloading the code from github

It is required to download the libedgetpu git to begin the build of the edgetpu shared library 

git clone https://github.com/google-coral/libedgetpu -b release-grouper
cd libedgetpu

Build and compile libedgetpu 

DOCKER_CPUS="k8" DOCKER_IMAGE="ubuntu:18.04" DOCKER_TARGETS=libedgetpu make docker-build

The generated shared objects should be in the "out" folder

Install script to install shared objects

Before running install script change line 181 to this 

"/usr/local/lib/libedgetpu.so.1.0"

Then you can run the install script 

sudo bash scripts/install.sh

CMake shared library compatibility command 

cd /usr/local/lib
sudo ln -s libedgetpu.so.1 libedgetpu.so

Example of adding libedgetpu to CMake project 

target_link_libraries(${PROJECT_NAME}
	PRIVATE libedgetpu.so
)

Install command for header file 

sudo cp tflite/public/*.h /usr/local/include
