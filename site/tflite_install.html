Installing TensorFlowLite C API from source

Installing Linux dependencies

Gonna need Google bazel. These are the preferred commands to install it 

wget https://github.com/bazelbuild/bazelisk/releases/download/v1.17.0/bazelisk-linux-amd64
mv bazelisk-linux-amd64 bazel
chmod +x bazel
mv bazel ~/.local/bin

Cloning TensorFlow from source (version 2.5.0)

To see all available versions, click here 

git clone https://github.com/tensorflow/tensorflow -b v2.5.0

Installation process 

cd tensorflow/tensorflow/lite
bazel build -c opt //tensorflow/lite/c:tensorflowlite_c
cp ../../bazel-bin/tensorflow/lite/c/libtensorflowlite_c.so /usr/lib

Cannot forget about the header files too! 

sudo mkdir -p /usr/local/include/tensorflow/lite/c
sudo cp tensorflow/lite/c/c_api.h /usr/local/include/tensorflow/lite/c/c_api.h
sudo cp tensorflow/lite/c/c_api_experimental.h /usr/local/include/tensorflow/lite/c/c_api_experimental.h
sudo cp tensorflow/lite/c/c_api_types.h /usr/local/include/tensorflow/lite/c/c_api_types.h
sudo cp tensorflow/lite/c/common.h /usr/local/include/tensorflow/lite/c/common.h
